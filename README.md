# Электронное меню кофейни

## Бизнес-проблема

Кофейня не хочет тратить бюджет на постоянную печать и обновление бумажного меню.
Решение — электронное меню, которое открывается по QR-коду на личном устройстве клиента.

## Функциональность

- Адаптивное веб-меню (работает на телефоне, планшете, ноутбуке)
- Стартовая страница с выбором языка: **RU / EN**
- Экран категорий: кофе, чай, десерты (каждая категория с картинкой и названием)
- Экран позиций: картинка, состав, цена
- Навигация "назад" и "домой", чтобы пользователь не терялся в структуре меню

## Технологии

- **Frontend:** HTML + CSS + JavaScript (без фреймворков)
- **Backend:** Node.js + Express
- **База данных:** SQLite (файл `menu.db`)
- Обмен данными по JSON через REST API (`/api/categories`, `/api/items`)
